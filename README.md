Se o programa demorasse 10 segundos para finalizar e a tela ficasse totalmente parada, o usuário provavelmente pensaria que o sistema travou ou que houve um erro.

A visibilidade de status, como as mensagens de "[PROCESSANDO]", melhora a Experiência do Usuário (UX) porque:
- Informa ao usuário que o sistema está trabalhando
- Reduz a incerteza e a ansiedade
- Evita que o usuário interrompa o processo ou reinicie o sistema
